# NYC House Prices Analysis

This project was developed by **Murat Berke I.** to analyze real estate prices in New York City. It focuses on boroughs, property types, and property features to provide actionable insights about the housing market.

## Project Overview

The goal of this project is to clean, explore, and visualize NYC housing data, highlighting trends and patterns in property prices. Key insights include median prices by borough and property type, top locations by price and price per square foot, and price trends based on the number of bedrooms.

## Features

- **Data Cleaning:**  
  - Removed irrelevant property types (e.g., "Land for sale").  
  - Filtered out unrealistic values for price, bedrooms, and bathrooms.  
  - Corrected borough names and handled missing data.

- **Analysis:**  
  - Median price by borough and property type  
  - Median price per square foot  
  - Top 5 most expensive boroughs and locations by price  
  - Price trends vs number of bedrooms

- **Visualizations:**  
  - Bar charts, line plots, and scatter plots using Matplotlib  

## Technologies Used

- **Python** – for scripting and data analysis  
- **Pandas** – for data cleaning, filtering, and aggregation  
- **Matplotlib** – for visualizing data
